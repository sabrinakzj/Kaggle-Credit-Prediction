# Home Credit Default Risk Prediction

[![Python](https://img.shields.io/badge/Python-3.7%2B-blue)](https://www.python.org/)
[![Kaggle](https://img.shields.io/badge/Kaggle-Competition-20BEFF)](https://www.kaggle.com/competitions/home-credit-default-risk)

A credit default risk prediction model built for the **Home Credit Default Risk** Kaggle competition.  
This project leverages machine learning to predict whether a loan applicant will default in the future, using historical loan application data to support more informed credit decisions.

---

## 📌 Project Background

Home Credit is an international financial institution that provides loans to people without traditional banking histories. Since conventional credit scoring methods don't cover this population, alternative data (application information, repayment records, etc.) is needed to build accurate risk prediction models.

**Competition Output**: Bronze Medal of the competition

## 🚀 Quick Start

### 1. Prerequisites

Ensure you have Python 3.7 or higher installed.

```bash
git clone https://github.com/sabrinakzj/Kaggle-Credit-Prediction.git
cd Kaggle-Credit-Prediction
```

### 2. Data Preparation

Download the dataset from the Kaggle competition page.
Place the data files in a data/ folder under the project directory (or adjust the path in the notebook accordingly).

### 3. Run the Notebook

Launch Jupyter Notebook or Jupyter Lab, open HomeCredit_Train.ipynb, and execute all cells sequentially.

### 4. If not run,read the article

The article introduces what codes do for the model and claims the useful findings.
